## Getting Started

1. Run `npm install`
2. Run `npm run dev`

## Supabase Setup

1. Create a Supabase project.
2. In the Supabase SQL Editor, run `supabase_schema.sql`.
3. Create an admin user in Supabase Auth (email/password).
4. Ensure `.env` has:
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_ANON_KEY`
   - `VITE_ADMIN_EMAIL`
